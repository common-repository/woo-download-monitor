	Monitor each download for your WooCommerce downloadable product.

== Description ==

If you are running downloadable WooCommerce store this plugin may help you lots to track downloads, who is downloading your product, details data for downloadable  product, date time, userid, product id, order id and etc.
You can also display downloadable product list on single product page with download link, user who already bought will able to download from link. otherwise no user will access to download.

### PickPlugins Download Monitor for WooCommerce by [https://www.pickplugins.com](https://www.pickplugins.com/?ref=wordpress.org)

[Live demo](http://www.pickplugins.com/demo/woo-download-monitor/)


### Plugin Features


<strong>Status for each download.</strong>
track each file download from your WooCommerce website. under dashboard you will able to see list of download via "Download Records" page. each record will display product thumbnail, product title, file name, product price, order id, user name, time & date.

<strong>Allow access by user role</strong>
You can set any product for download access by specific user role, so user can access for download file without purchasing. other user will not able to access download.

<strong>Allow public access</strong>
You can allow all user even logged out user will able to download file without purchasing the product.

<strong>Restrict user</strong>
You can also restrict any use for access to download file.


<strong>Translation</strong>

Plugin is translation ready , please find the 'en.po' for default translation file under 'languages' folder and add your own translation. you can also contribute in translation, please contact us http://www.pickplugins.com/contact/



### Shortcodes

You can display anywhere downloadable files via shortcode

`[wdm_downloadable_files id="123"]`

* `id` id product id.




== Installation ==

1. Install as regular WordPress plugin.<br />
2. Go your plugin setting via WordPress Dashboard and find "<strong>PickPlugins Download Monitor for WooCommerce</strong>" activate it.<br />
